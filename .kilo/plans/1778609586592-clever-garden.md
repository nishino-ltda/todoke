## Observações

- Nenhum parceiro real possui coordenadas no banco; as coordenadas são geradas deterministicamente via `fmod($id, 10)` para evitar variação a cada carregamento.
- O carrinho já persiste em localStorage (store `cart.js`), portanto itens adicionados sem autenticação são preservados após o login e redirecionamento para o checkout.
- A rota `/customer/checkout` exige autenticação (middleware `auth`), mas como o fluxo de login é concluído antes do redirect, o usuário autenticado consegue acessá-la normalmente.
- A nova página `PartnerStore.vue` usa um layout público (não `CustomerLayout`) para não exigir autenticação.
- A rota `/store/{partner}` é pública (sem middleware `auth`).
